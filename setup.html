MY SANDA 配置向导

按照以下步骤配置你的 Supabase 项目

步骤 1：创建 Supabase 项目

  1. 访问 https://supabase.com
  2. 点击 "Sign Up" 注册账户
  3. 点击 "New Project" 创建新项目
  4. 填写项目名称（如 "my-sanda"）
  5. 设置数据库密码
  6. 选择地区（推荐选择离你最近的）
  7. 点击 "Create new project" 并等待完成

步骤 2：获取 API 密钥

  1. 进入你的项目
  2. 点击左侧菜单 "Settings"（设置）
  3. 选择 "API"
  4. 复制 "Project URL"（看起来像 https://xxxxx.supabase.co）
  5. 复制 "anon public" 密钥

步骤 3：关闭邮箱验证

  1. 在 Supabase 项目中，点击左侧菜单 "Authentication"
  2. 选择 "Providers"
  3. 找到 "Email" 提供商
  4. 关闭 "Confirm email" 开关
  5. 点击 "Save" 保存

步骤 4：配置项目文件

  1. 打开项目中的 js/app.js 文件
  2. 找到第 1-2 行的配置： 
    const SUPABASE_URL = 'https://your-project.supabase.co';
const SUPABASE_KEY = 'your-anon-key';
  3. 替换为你的实际值： 
    const SUPABASE_URL = 'https://xxxxx.supabase.co';
const SUPABASE_KEY = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
  4. 保存文件

步骤 5：启动项目

选择以下任一方式启动项目：

方式 A：Python（推荐）

python -m http.server 8000

然后在浏览器打开：http://localhost:8000

方式 B：Node.js

npm install
npm start

方式 C：Windows 用户

双击运行 start-server.bat 文件

✅ 完成！
