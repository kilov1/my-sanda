# 返回首页按钮功能实现说明

## 功能描述
在页面左上角 MY SANDA logo 右边添加了一个【返回首页】按钮。

## 实现的功能

### 1. ✅ 按钮位置
- **位置**：导航栏左上角，MY SANDA logo 右侧
- **样式**：🏠 返回首页（带房子图标）
- **颜色**：灰色文字，hover 时变为蓝色

### 2. ✅ 显示逻辑
- **在首页时**：按钮隐藏（hidden）
- **在子页面时**：按钮显示
  - 学习资料页面 ✓
  - 校园动态页面 ✓
  - 校园生活页面 ✓
  - 失物招领页面 ✓
  - 二手闲置页面 ✓
  - 提问求助页面 ✓
  - 登录/注册页面 ✓
  - 个人资料页面 ✓
  - 所有其他子页面 ✓

### 3. ✅ 点击功能
- 点击按钮后：
  - 跳转回首页
  - 首页分类按钮变为激活状态
  - 其他分类按钮取消激活状态
  - 返回首页按钮自动隐藏

## 代码修改

### HTML 修改（index.html）
```html

<button id="navHomeBtn" class="hidden text-gray-600 hover:text-blue-600 transition-colors text-sm font-medium px-3 py-1.5 rounded-lg hover:bg-blue-50">🏠 返回首页</button>
```

### JavaScript 修改（app.js）

#### 1. showPage 函数逻辑
```javascript
if (pageId === 'homePage') {
    toggleNavBackButton(false);
    toggleNavHomeButton(false);  // 首页时隐藏
    pageHistory = [];
} else {
    toggleNavBackButton(true);
    toggleNavHomeButton(true);   // 子页面时显示
}
```

#### 2. 返回首页按钮点击事件
```javascript
document.getElementById('navHomeBtn')?.addEventListener('click', () => {
    document.querySelectorAll('.category-btn').forEach(b => b.classList.remove('active'));
    const homeBtn = document.querySelector('.category-btn[data-page="homePage"]');
    if (homeBtn) homeBtn.classList.add('active');
    showPage('homePage');
});
```

#### 3. toggleNavHomeButton 函数
```javascript
function toggleNavHomeButton(show) {
    const navHomeBtn = document.getElementById('navHomeBtn');
    if (navHomeBtn) {
        if (show) {
            navHomeBtn.classList.remove('hidden');
        } else {
            navHomeBtn.classList.add('hidden');
        }
    }
}
```
